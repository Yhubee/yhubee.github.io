# Ubong Eno Nkereuwem — Human–Centered AI Profile

Static, GitHub Pages–ready site showcasing academic positioning, research agenda, and projects in Human–Centered AI, HCI, and Technical AI Product Management.

## Using on GitHub Pages
1. Ensure GitHub Pages is enabled for the repository and points to the default branch.
2. All site files live in the repository root; no build step is required.
3. Images are referenced from the `content/assets` directory.

## Structure
- `index.html` — landing page
- `about.html` — background, motivation, and academic formation
- `projects.html` — AI + HCI case studies
- `research.html` — research themes and applied threads
- `publications.html` — academic positioning and writing highlights
- `contact.html` — contact details and mentoring overview
- `styles.css` — shared styling
