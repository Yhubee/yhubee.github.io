LearnLink – Peer-to-Peer Skill Exchange Platform

Cash-free, community skill swaps built as a fast, full-stack MVP.

Role: Full-stack developer & product collaborator

Context: Course full-stack MVP at La Fosse

Domains: Peer-to-peer learning, community platforms, data-driven dashboards

Methods: Agile teamwork, UX design, full-stack implementation, data modelling, technical documentation

Overview

LearnLink pairs people who want to teach or learn skills nearby, letting them swap time instead of cash.

The MVP combined UX, Agile delivery, and a simple data model to prove the peer-to-peer learning loop.

Challenge

Traditional courses can be expensive or rigid. We needed a lightweight way for people to share skills locally without payment.

Stakeholder Insights

  • Learners want affordable, flexible access.
  • Skill-holders need a simple way to offer time.
  • Communities value informal, local learning.

Our Solution

A location-aware web app where users post skills to teach or learn, then connect for in-person or online swaps.

Key Features

  • Signup / login with profile setup.
  • Location-based skill matchmaking and requests.
  • Messaging, availability, and simple dashboards.
  • Responsive UI powered by a PostgreSQL backend.

Backlog Highlights

  • Ratings, swap history, and learning streaks.
  • QR logging and social profile sharing.
  • Mobile app version.

Technical Snapshot

  • Stack: Node.js, Express, Socket.io, PostgreSQL, HTML/CSS/JS.
  • Auth with hashed passwords; tested with Jest and Supertest.
  • Deployed to Render / Netlify with feature-branch Git workflow.
