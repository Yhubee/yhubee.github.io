Project Demo – Beyond STEM

Overview

Beyond STEM is a quick, game-based web app for non-STEM lessons, built to spark curiosity with short history fact cards.

The MVP combined UX design, Agile delivery, and a lightweight SQL backend.

Challenge

Students were tuning out of non-STEM subjects. We needed a playful format that could fit into short lessons.

Stakeholder Insights

  • Students want novelty and quick wins.
  • Teachers need low-friction content.
  • Parents and leaders want visible engagement.

Our Solution

A card-based game with unusual facts, images, and scoring to keep teens clicking while learning.

Key Features

  • Signup / login and saved progress.
  • Fact cards with bite-sized history content.
  • SQL-backed data and responsive frontend.

Technical Snapshot

  • Stack: Node.js, Express, SQL, HTML/CSS/JS.
  • Auth with hashed passwords; tested with Jest and Supertest (78% coverage).
  • Deployed on Render / Netlify with feature-branch Git workflow.
